Welcome to the repo site.

I'm basically sick of the British Fencing calendar not having all events on it. This idea isn't perfect but its a way around certain problems:

  • Don't want to / aren't able to pay BF to put your event on the Calendar
  • Don't have the skill set to handle creating a website for your event
  • Online payments are tricky and you don't want to touch them with a barge pole

The idea is that this should make running a competition easier (in terms of online presense) and should make it easier for fencers to enter.

Basic premise

  1. Competition organiser creates event (name, venue, type, category, entry fee etc)
  2. Fencer creates their profile (name, weapons fenced, license number, location etc
  3. Fencer searches for competitions (either by weapons offered, location, celtic passport, anything we can think of
  4. Fencer enteres competition
  5. Competition organiser can see live update of entry lists, download list, see who has valid licenses etc
